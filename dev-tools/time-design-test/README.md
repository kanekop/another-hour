# Time Design Test UI

開発者向けの包括的なTime Design Modesテストインターフェースです。

## クイックスタート
1. ブラウザで `/dev-tools/time-design-test/` にアクセス
2. 左パネルからモードを選択
3. 設定を調整してリアルタイムで確認

## 詳細仕様
完全な技術仕様とUI/UX設計については、[こちらの仕様書](../../docs/ui-specifications/time-design-test-ui-spec.md)を参照してください。

## 主な機能
- 全モードの動作確認
- リアルタイムデバッグ
- パフォーマンス計測
- 設定のエクスポート/インポート
